# 🚀 CI SUM PYTHON - FastAPI + Docker + CI/CD

Projeto desenvolvido para a disciplina de DevOps com o objetivo de:

- Evoluir um script Python para uma API com FastAPI
- Criar testes automatizados com Pytest
- Containerizar a aplicação com Docker
- Implementar build e push automático com GitHub Actions
- Publicar imagem no Docker Hub

---

## 📦 Tecnologias utilizadas

- Python 3.13
- FastAPI
- Uvicorn
- Pytest
- Docker
- GitHub Actions
